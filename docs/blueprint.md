# **App Name**: Toolbox AI

## Core Features:

- Tool Aggregation: Provide access to the specified list of 20 free tools, focusing on frequently used utilities, and AI enhanced tools such as text summarizer, AI Essay writer, grammar checker, and plagiarism checker.
- Contextual AI Assistance: Offer optional AI assistance to guide users through each tool, by having the LLM function as a tool that knows how to make efficient use of all the other tools.
- PDF Conversion: Enable users to convert PDF files to Word documents and vice versa. Implement a fast conversion process to maintain the quality of the documents.
- Image Optimization: Offer tools to compress images and convert images to text using OCR, ensuring minimal quality loss and efficient text extraction.
- Content Creation: Include tools for QR code generation, text summarization, and a meta tag generator to aid in quick content creation and SEO optimization.
- Utility Tools: Incorporate essential utilities like an age calculator, unit converter, currency converter, JSON formatter, and code minifier to cater to a broad audience.
- AdMob Integration: Seamlessly integrate Google AdMob to monetize the app without disrupting the user experience, ensuring compliance with AdMob policies.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey professionalism and trustworthiness, without being too playful.
- Background color: Light Lavender (#E6E6FA) for a soft, non-intrusive backdrop.
- Accent color: Royal Purple (#7851A9) to highlight key actions and elements.
- Body and headline font: 'Inter', a sans-serif font, offering a modern, clean aesthetic suitable for various screen sizes.
- Use a set of minimalist icons that clearly represent each tool's function, in a consistent style throughout the app.
- Design a clean, intuitive layout optimized for mobile devices, ensuring each tool is easily accessible.
- Incorporate subtle animations to enhance the user experience, such as loading indicators and transitions between tools, to provide a smooth feel.
